Welcome to the L.it X Engine! v.0.2
The Livingfor.it Engine is an open source ASCII Game engine written in Java. Easy, intuitive and most importantly, powerfull!

Features

- Draw out raw pixels at coordinates.
- Plotter - Draw out your art, and import it into LitEngine
- Get user input (Strings and Input)
- Draw rectangles and circles
- Print animated text, perfect for RPGs.
- Backwards combatibility, your game will work in all future versions of LitEngine.
- Set your own resolution.
- More features in the documentation!

Planned features

- JFrame integration
- Keypress input
- Multithreaded heartbeat
- Audio support

Quickstart guide:

//Initiate LitEngine
LitEngine.start("border");

//Get input
String string = LitEngine.inputString();

//Print out text
LitEngine.print(x, y, string);
